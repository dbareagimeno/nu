# P40 · Registry central de plugins (índice curado con búsqueda/publicación, más allá del gestor git de P4→ADR-025)

**Dónde se pospuso.** [ADR-025](../decisions/adr/adr-025-reposicionamiento-motor-de-harnesses.md), pieza 3 / [auditoría externa 2026-07-18](../audits/auditoria-externa-concepto-2026-07-18.md)

**Por qué.** El gestor sobre git (repos + tags/SHA + lockfile) cubre la distribución sin operar infraestructura propia ni convertirse en gatekeeper; un registry antes de tener autores es un edificio vacío

**Disparador de reapertura.** Una masa real de plugins de terceros en git (decenas) donde el descubrimiento o la confianza (¿quién publica qué?) duelan de verdad; cf. P38 (repo aparte para el ecosistema)
